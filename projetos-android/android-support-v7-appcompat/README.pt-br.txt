Projeto de biblioteca, incluindo ActionBar de compatibilidade.

Isso pode ser usado por um projeto Android para fornecer
acesso ao ActionBar em aplicativos em execução na API 7+.

Tecnicamente, não há fonte, mas a pasta src é necessária
para garantir que o sistema de construção funcione. O conteúdo é realmente
localizado em libs / android-support-v7-appcompat.jar.
Os recursos acompanhantes também devem ser incluídos no aplicativo.
